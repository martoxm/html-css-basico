# HTML e CSS Básico

Repositório com páginas simples criadas para praticar estruturação de conteúdo com HTML5 e estilização com CSS3.

## Conteúdos praticados
- Estrutura básica de páginas HTML
- Tags semânticas
- Listas, tabelas e formulários
- Imagens e links
- Estilização com CSS
- Layout básico

## Objetivo
Aprender os fundamentos do desenvolvimento web e criar páginas organizadas e responsivas.

## Tecnologias
- HTML5
- CSS3
